  • Template A CSS class-based, jQuery event driven front-end framework for making sweet interfaces.

  • jQuery.event.tap
    jQuery special event tap for speeding up touch interfaces.

    jQuery special events movestart, move and moveend for tracking touch and mouse moves, throttled to browser animation frames.
